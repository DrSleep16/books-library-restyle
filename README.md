## Как использовать скрипт для скачивания книг с сайта tululu.org

Этот скрипт позволяет скачивать книги с сайта [tululu.org](https://tululu.org), а также извлекать информацию о них, такую как название, автор, жанры и комментарии. Скрипт поддерживает опциональные аргументы командной строки, которые позволяют указать диапазон страниц для скачивания.

### Установка и использование

1. Скопируйте скрипт `main.py` в свой рабочий каталог.

2. Откройте командную строку (терминал) и перейдите в каталог, где находится скрипт `main.py`.

3. Для скачивания книг из диапазона страниц используйте следующий синтаксис команды:

   ```bash
   python main.py <start_id> <end_id>
   ```

   Где `<start_id>` и `<end_id>` - это номера страниц, с которых начать и закончить скачивание книг соответственно. Например:

   ```bash
   python main.py 1 10
   ```

   Эта команда скачает книги с 1-й по 10-ю страницу сайта tululu.org.

4. Скачанные книги будут сохранены в каталоге `books/` в формате текстовых файлов `.txt`. Изображения обложек книг будут сохранены в каталоге `images/`.

### Дополнительная информация

- Для каждой скачанной книги, скрипт также извлекает и выводит информацию о ней, включая название, автора, жанры и комментарии (если они есть).

- Все изображения обложек сохраняются в формате `.jpg`, а если обложка отсутствует, то сохраняется файл `nopic.gif`.
